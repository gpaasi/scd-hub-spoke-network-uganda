# SCD Hub-and-Spoke Network Analysis for Uganda

This repository contains the analysis outputs, supplementary documents, figures, and reproducibility scripts for the manuscript:

**Linking newborn screening to essential sickle cell disease care in Uganda: a spatial hub-and-spoke network analysis of access, readiness, and strengthening priorities**

## Study aim

The paper develops a national newborn screening-to-sickle cell disease (SCD) care network for Uganda by linking accepted geocoded newborn screening (NBS) spokes to eligible Health Centre IV-and-above SCD care hubs. The analysis assesses route feasibility, receiving-hub readiness, hub workload, weak linkages, service gaps, and strengthening priorities.

## Key manuscript outputs

The uploaded analysis data support the following manuscript result domains:

1. Analytic facility layers and SCD service-readiness profile
2. Scenario-based spoke-to-hub distance structure
3. Nearest-hub assignment under alternative eligibility scenarios
4. Route feasibility in the primary assigned network
5. Hub workload and route-risk concentration
6. Facilities prioritised for SCD service strengthening
7. Service-readiness gaps among strengthening facilities
8. Spatial implementation interpretation

A detailed crosswalk is available at `data/metadata/results_crosswalk.csv`.

## Reproduce the audit checks

Create a Python environment and install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Run the validation script:

```bash
python scripts/validate_results.py
```

This writes:

```text
outputs/validation/validation_report.md
outputs/validation/validation_report.csv
```

## Data interpretation note

The CSV files are analysis outputs for an implementation-planning model. Straight-line distances represent planning distances, not road-network travel distances. Modelled routes represent potential screening-to-care linkages, not observed patient movement. Readiness indicators represent measured service availability, not validated service quality or effective coverage.
